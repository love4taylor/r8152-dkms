Realtek USB Ethernet Driver

r8152 DKMS

面向 Debian 与 Ubuntu 的签名软件源。安装一次，DKMS 会在内核安装或升级时自动重新构建驱动模块。

Installation

三步完成安装

所有软件包元数据均由独立仓库密钥签名，密钥仅用于这个软件源。

r8152.deb.love4taylor.com
01

安装仓库签名密钥

安装 HTTPS 下载工具，将二进制公钥保存到系统 keyring；APT 只会用它验证此仓库。

Shell 
sudo apt update
sudo apt install --yes ca-certificates curl
sudo install -d -m 0755 /usr/share/keyrings
curl -fsSL https://r8152.deb.love4taylor.com/keyrings/r8152-archive-keyring.gpg \
  | sudo tee /usr/share/keyrings/r8152-archive-keyring.gpg >/dev/null
02

添加 APT 软件源

通用的 any suite 不依赖 Debian 或 Ubuntu 的发行版代号。

Shell 
echo "deb [signed-by=/usr/share/keyrings/r8152-archive-keyring.gpg] https://r8152.deb.love4taylor.com any main" \
  | sudo tee /etc/apt/sources.list.d/r8152.list
03

安装驱动包

安装当前内核头文件，并让 DKMS 为当前内核构建 r8152 模块。

Shell 
sudo apt update
sudo apt install linux-headers-$(uname -r) r8152-dkms
!

这是 DKMS 源码包

软件包不包含预编译的 .ko。安装时会在本机编译模块；切换正在使用的网卡驱动可能暂时中断网络连接。

确认安装状态

安装完成后，可检查 DKMS 记录和模块信息。

dkms status | grep r8152
modinfo r8152
移除软件包与软件源
Shell 
sudo apt remove r8152-dkms
sudo rm -f /etc/apt/sources.list.d/r8152.list
sudo rm -f /usr/share/keyrings/r8152-archive-keyring.gpg
sudo apt update
